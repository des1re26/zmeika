# README.md

## Змейка (Snake Game)

Классическая игра «Змейка» с тремя игровыми режимами, графическим интерфейсом на SFML, системой настроек и сохранением рекордов.

---

## Возможности

- **Три игровых режима:**
  - *Classic* – классическая змейка, стены убивают
  - *Wrap* – телепортация через стены
  - *Swap* – при съедании еды голова и хвост меняются местами

- **Графический интерфейс:**
  - Главное меню с навигацией
  - Меню выбора режима с описанием
  - Меню настроек (размер клетки, полноэкранный режим, VSync, FPS)
  - Меню паузы по клавише Escape

- **Система рекордов:**
  - Отдельный рекорд для каждого режима
  - Автоматическое сохранение в файл records.txt

---

## Управление в игре

| Клавиша | Действие |
|---------|----------|
| ↑ | Движение вверх |
| ↓ | Движение вниз |
| ← | Движение влево |
| → | Движение вправо |
| Esc | Пауза / выход из меню |
| R | Перезапуск после Game Over |

**Навигация по меню:**
| Клавиша | Действие |
|---------|----------|
| ↑ / ↓ | Перемещение между пунктами |
| Enter | Выбор пункта |
| Esc | Выход / возврат |

**В меню настроек:**
| Клавиша | Действие |
|---------|----------|
| ← / → | Изменение значения параметра |
| Enter | Применить и выйти |

## Описание режимов

| Режим | Описание |
|-------|----------|
| **Classic** | Столкновение со стеной или с собственным телом приводит к завершению игры. Каждая съеденная еда приносит 10 очков. |
| **Wrap** | При выходе за границу змейка появляется с противоположной стороны. Проверка столкновения с собой начинается со второго сегмента. |
| **Swap** | При съедании еды змейка разворачивается (голова становится хвостом). За разворот начисляются дополнительные 5 очков. |

## Настройки

| Параметр | Описание | Диапазон |
|----------|----------|----------|
| Cell Size | Размер клетки в пикселях | 20-50 (шаг 5) |
| Fullscreen | Полноэкранный режим | ON / OFF |
| VSync | Вертикальная синхронизация | ON / OFF |
| FPS Limit | Ограничение частоты кадров | 30-240 (шаг 15) |

## Требования для сборки

- Компилятор с поддержкой C++17 (Visual Studio 2022, GCC, Clang)
- Библиотека SFML 2.6 или выше
- Файл шрифта `ArialRegular.ttf` (любой .ttf шрифт)



## Архитектура

Проект построен на принципах объектно-ориентированного программирования:

- **GameApplication** – управляет общим потоком приложения
- **Game** – базовый класс с виртуальными методами для переопределения
- **Wrap / Swap** – наследуются от Game и переопределяют поведение
- **Snake** – отдельный класс, инкапсулирующий логику змейки
- **Renderer** – вся графика вынесена в отдельный класс
- **Menu** – иерархия классов меню с общим базовым классом

## Устранение неполадок

| Проблема | Решение |
|----------|---------|
| Не отображается текст | Убедитесь, что файл `ArialRegular.ttf` находится в папке с программой |
| Ошибка при запуске | Скопируйте все DLL-файлы SFML в папку с исполняемым файлом |
| Не сохраняются рекорды | Проверьте права на запись в папке с программой |
